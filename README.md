# Snake Game

一个用 C++ 和 Qt 实现的贪吃蛇游戏，支持双人对战、AI 对手、地形系统（墙 / 加速带 / 传送门）和多种预置地图。

## 背景

个人学习 C++ 和 Qt 时的练手项目，在经典贪吃蛇基础上扩展了多玩家、简单地形、传送门、重生机制等。代码结构比较直接，可供初学者参考。

## 主要功能

- **1 ~ 2 名真人玩家**（按 `P` 实时切换）
- **AI 玩家**（2 人模式下按 `C` 将 2P 切换为 AI，AI 使用 A* + BFS 寻路）
- **4 种预置地图**（按 `1` `2` `3` `4` 实时切换并重启对局）
- **地形系统**：
  - 墙 – 不可通过
  - 加速带 – 蛇移动速度加倍
  - 传送门 – 进入后传送到对应出口
- **其他操作**：空格暂停 / 继续，WASD 控制 1P，方向键控制 2P

## 依赖环境

- C++17（或更高）
- Qt 5 / Qt 6（使用 QPainter 绘图、QTimer 驱动游戏循环）
